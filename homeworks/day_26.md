## Реализация (демо на Ollama)

Для демо выбран `Ollama` как самый быстрый вариант локального запуска на macOS.
Модель поднимается локально и доступна по HTTP API (`http://localhost:11434`).

Для удобства запустил на ВМ
```bash
ssh -l kdsoko -i ~/.ssh/id_ed25519_yandex 81.26.185.77
```

1. Установить Ollama:

```bash
curl -fsSL https://ollama.com/install.sh | sh

WARNING: No NVIDIA/AMD GPU detected. Ollama will run in CPU-only mode.
```

2. Запустить Ollama-сервер:

```bash
ollama serve
```

3. В отдельном терминале скачать и подготовить модель:

```bash
ollama pull qwen2.5:3b
```

4. Проверка, что модель установлена:

```bash
ollama list
NAME          ID              SIZE      MODIFIED      
qwen2.5:3b    357c53fb659c    1.9 GB    3 seconds ago 
```

Ожидаемо в списке есть `qwen2.5:3b`.

## Проверка 1: локальный запуск модели (CLI)

```bash
ollama run qwen2.5:3b "Ответь одним предложением: что такое локальная LLM?"
Локальная LLM представляет собой машинную learning модель запущенную на специализированной инфраструктуре для ускорения процесса 
обучения и повышения производительности.

```

Ожидаемый результат: модель отвечает в консоли, без обращения к внешнему API.

## Минимум 3 запроса разной сложности

### Запрос 1 (простой)

```bash
ollama run qwen2.5:3b "Объясни в 2 предложениях, что такое токены в LLM."
Токены в ланг-uae-генерирующих моделях (LLM) представляют собой базовые единицы текста, которые модели используют для генерации 
последовательностей предложений или ответов на запросы пользователя. Это позволяет модель расщеплять входные данные на более мелкие 
части для анализа и понимания.
```
