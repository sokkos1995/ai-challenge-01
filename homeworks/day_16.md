🔥 День 16. Подключение MCP

Установите MCP SDK / клиент (или поднимите MCP-сервер, если используете локальный вариант)

Сделайте минимальный код, который:

👉 устанавливает MCP-соединение
👉 получает от MCP список доступных инструментов

Проверьте:

👉 соединение устанавливается
👉 список инструментов корректно возвращается

Результат:

Код, который подключается к MCP и выводит список доступных инструментов

## Демо

Для задания реализованы 2 файла:

- `homeworks/src/day_16_server.py` - локальный MCP-сервер
- `homeworks/src/day_16_client.py` - клиент, который подключается к серверу и запрашивает список инструментов

Запуск демо:

```bash
python3 homeworks/src/day_16_client.py
```

Ожидаемый вывод:

```text
1. Запускаем локальный MCP-сервер и открываем соединение...
2. Соединение открыто. Создаем MCP-сессию...
3. Выполняем MCP initialize()...
4. MCP-соединение установлено.
   Имя сервера: day-16-local-server
   Версия протокола: 2025-11-25
5. Запрашиваем у сервера список доступных инструментов...
6. Получено инструментов: 2
   1. hello - Возвращает простое приветствие.
   2. add - Складывает два числа и возвращает результат.
```

Проверка условий задания:

- `соединение устанавливается` - это подтверждается строкой `4. MCP-соединение установлено.`
- `список инструментов корректно возвращается` - это подтверждается строками `6. Получено инструментов: 2` и списком инструментов `hello`, `add`
